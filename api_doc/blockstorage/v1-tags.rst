.. swagger:tag:: volumes
   :synopsis: Volumes

   The ``snapshot_id`` and ``source_volid`` parameters specify the ID
   of the snapshot or volume from which the volume originates. If the
   volume was not created from a snapshot or source volume, these
   values are null.
